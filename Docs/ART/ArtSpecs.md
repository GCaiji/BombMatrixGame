## 特效规范
1. **爆炸特效参数**：
   - 粒子数量：50-80
   - 持续时间：1.2秒
   - 颜色渐变：红 → 橙 → 透明
2. **地形破坏效果**：
   - 使用溶解Shader实现边缘腐蚀效果
   - 参数示例：
     ```hlsl
     _DissolveAmount("Dissolve Amount", Range(0,1)) = 0.5
     ```
